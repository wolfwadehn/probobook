PART LOADING  (part-loading.adoc)
────────────────────────────────────────────────────────────────────
Functional View
  Phase 1 — Application Bootstrap
  Phase 2 — Window and OpenGL Surface Creation
  Phase 3 — Scene Initialisation (OnLuxReady)
  Phase 4 — Robot Scene Construction
      4a  Mechanism loading (mechanism.curl → kinematic tree + meshes)
      4b  Node lookup (Tip, S L U R B T)
      4c  Joint limits (JMin/JMax → mMin[], mMax[])
      4d  IK solver construction (DH parameters table)
      4e  Obstacle box (Poly → Extrude → OBBTree → Mesh3VN → XfmVN)
      4f  Link OBB trees (CMesh / Mesh fallback, local space)
      4g  Gripper transform node
      4h  Home pose and first IK call
      4i  Script timer (DispatcherTimer 500 ms)
      4j  Overlay nodes (InfoVN, TcpVN)
      4k  Scene graph assembly (Root GroupVN)
  Phase 5 — UI Population (CreateUI)
  Phase 6 — Steady-State Operation
